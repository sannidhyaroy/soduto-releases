Version 2.0.2

Enhancements:

  • Auto-check for updates button added in Preferences.

Improvements:

  • Soduto wouldn't explicitly request for notification permissions, if it wasn't already authorized to push notifications in macOS. We've fixed this issue in this release.
  • There was an issue with Auto-copy OTP from Google Messages to clipboard where Soduto would copy the OTP along with the hidden/invisible characters between the OTP digits which resulted in failed OTP validation.
  • Notification app icons were relocated & reorganized.

Known bugs:

The Share Extension is a nightly feature and hence buggy & doesn't provide support sharing multiple files yet. Select only one file and use the share extension. Soduto Share won't appear in the share menu if multiple files are selected.

WORKAROUND:

To share multiple files, use the option to Send Files from the menu bar app options or drag & drop multiple selected files to the menu bar icon.
