Version 3.0.0

⚠️ BREAKING CHANGES

Support for macOS 10.x and earlier has been dropped. [c424476]
The minimum supported macOS version is now macOS 11.0 (Big Sur).

Soduto is a decade-old project and accumulated a large amount of deprecated and legacy code over time. Replacing these components with modern APIs while keeping older macOS versions functional has become increasingly difficult and unsustainable. Additionally, macOS 10.x is more than six years old. Given Apple’s long software support lifecycle, systems still running these versions are typically very outdated, making ongoing maintenance, testing, and stability harder to guarantee.

Untangling and replacing these legacy dependencies will require multiple development iterations, as Soduto is deeply integrated with deprecated components and legacy Objective-C code, due to having seen little active development for nearly a decade.

Anyone interested in maintaining or extending support for older macOS versions is free to fork the project and continue development.

Known Bugs

  • The notification payload is usually sent only once by KDE Connect for each notification ID. Dismissing a notification and using the Request Notifications option may result in missing notification icons. This also affects messaging apps like WhatsApp that reuse notification IDs. This is a KDE Connect limitation with no clean workaround currently.
  • Some apps, such as Ghostty, are known to capture keyboard input, preventing termination of the Control Remote Device function even after pressing Option + Escape.

Workaround

  • To forcefully exit Control Remote Device, go to the App Info page of KDE/Zorin Connect on your Android device and tap Force Stop.
